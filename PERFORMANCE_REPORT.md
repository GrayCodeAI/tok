## Performance Recommendations

### For Development (Default)
```bash
tokman --preset balanced
```
- Good balance of speed and compression
- ~400µs latency for typical inputs

### For Maximum Compression
```bash
tokman --preset full --mode aggressive
```
- Best for CI/CD logs, large outputs
- 90%+ token reduction

### For Speed-Critical
```bash
tokman --preset fast
```
- Minimal layers, fastest processing
- 20-40% token reduction
