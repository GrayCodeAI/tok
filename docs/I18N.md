## Language Files

```
locales/
├── en.toml  # English (reference)
├── fr.toml  # French
├── zh.toml  # Chinese (Simplified)
├── ja.toml  # Japanese
├── es.toml  # Spanish
├── de.toml  # German
└── ko.toml  # Korean
```

## Usage

```bash
# Use Japanese
tokman --lang ja --help

# Use Chinese
tokman --lang zh stats
```

## Contributing Translations

1. Copy `locales/en.toml` to `locales/xx.toml`
2. Translate all values (keep keys unchanged)
3. Submit PR with your changes
4. We'll verify and merge

### Translation Guidelines

- Keep placeholders like `{version}`, `{error}`, `{action}` unchanged
- Maintain similar message length (UI layout)
- Use natural language (not literal translation)
- Test with actual commands before submitting
