# Homebrew Tap for TokMan

Token-aware CLI proxy with advanced quality analysis.

## Installation

```bash
# Add this tap
brew tap GrayCodeAI/tokman

# Install TokMan
brew install tokman
```

## Usage

```bash
tokman --version
tokman init -g
tokman git status
tokman gain
```
